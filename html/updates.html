2026年3月27日 - 网站全面重新设计

概述：对网站进行了全面的现代化改造，包括视觉设计、响应式布局和用户体验的全面提升。

HTML 改进

  • 新增 响应式视口元标签，支持移动设备
  • 新增 页面描述元标签，提升SEO
  • 改进 更新页面标题为"Shen Zhihao - Research Notes"
  • 修复 QR代码图片属性语法
  • 改进 移除不必要的<br/>标签

CSS 重新设计

  • 新增 现代CSS重置和盒模型设置
  • 新增 平滑滚动效果
  • 新增 渐变背景替换固定背景图片
  • 新增 粘性导航栏和毛玻璃效果
  • 新增 动画下划线悬停效果
  • 新增 渐变标题背景和文字阴影
  • 新增 响应式媒体查询，支持所有屏幕尺寸
  • 新增 Flexbox/Grid布局系统替换旧的浮动布局
  • 新增 交错淡入动画效果
  • 新增 现代卡片设计和悬停动画
  • 新增 侧边栏悬停效果和平滑过渡

  • 改进 专业配色方案和渐变效果

  • 新增 所有交互元素的悬停效果

  • 改进 完全自适应响应式布局
  • 改进 移动优先设计方法
  • 改进 触控友好的界面

  • 技术 使用现代CSS特性（Grid、Flexbox、CSS变量）
